Настройка API ключа
Шаг 1: Получение ключа [OpenWeatherMap](https://aviationstack.com/)
Перейди на сайт https://aviationstack.com/
Нажми Sign Up и зарегистрируйся (России нет можно указать другую страну и выбрать -  I am student)
Включи VPN
Перейди в раздел API Keys
Скопируй свой ключ (длинная строка из букв и цифр)
Важно: Активация ключа занимает 10–15 минут.
Если сразу после получения ты увидишь ошибку 401 Unauthorized — просто подожди немного.
Шаг 2: Создание файла gradle.properties
В корне проекта (рядом с папкой app/) создай (если нет) файл gradle.properties
Добавь в него следующую строку:

AVIATIONSTACK_API_KEY=твой_ключ_от_openweathermap

Шаг 3: Добавление в .gitignore
Убедись, что файл gradle.properties находится в .gitignore, чтобы ключ не улетел в репозиторий
