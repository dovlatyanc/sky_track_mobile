# ✈️ Sky Track

Приложение для отслеживания рейсов в реальном времени.

##  Технологический стэк

- **Kotlin** — основной язык программирования
- **Jetpack Compose** — декларативный UI фреймворк
- **Material 3** — дизайн система
- **Koin** — Dependency Injection
- **Retrofit + OkHttp** — работа с сетью
- **Room** — локальная база данных
- **Coroutines + Flow** — асинхронность
- **Accompanist SwipeRefresh** — pull-to-refresh
- **MVVM / MVI** — архитектурный паттерн

##  Функциональность

-  Просмотр списка рейсов из API aviationstack.com
-  Детальная информация о рейсе
-  Pull-to-refresh для обновления списка
-  Сохранение просмотренных рейсов (Room)
-  Избранные рейсы
-  Сортировка: просмотренные вверху списка
-  Визуальное выделение просмотренных рейсов
-  Возможность поделиться информацией о рейсе
-  ProgressBar во время загрузки

##  Архитектура

- **data** — слой данных (модели, репозитории, API, база данных)
- **presentation** — слой представления (UI, компоненты, экраны)
- **di** — внедрение зависимостей (Koin)
